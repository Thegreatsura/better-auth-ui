The `@better-auth-ui/react` package ships the data layer behind every Better Auth UI component. It's built on [TanStack Query](https://tanstack.com/query) and exposes two kinds of primitives per endpoint:

- **Hooks** ‚Äî `useSession`, `useListAccounts`, ‚Ä¶
- **Options factories** ‚Äî `sessionOptions`, `listAccountsOptions`, ‚Ä¶

Access a query's key via the options object: `sessionOptions(authClient).queryKey`.

## Prerequisites

Wrap your app with a `QueryClientProvider` above `<AuthProvider>`.

```package-install
@tanstack/react-query
```
