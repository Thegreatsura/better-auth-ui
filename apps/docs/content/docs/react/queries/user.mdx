---
title: useUser
description: Retrieve the current authenticated user.
---

Thin wrapper over [`useSession`](/docs/react/queries/session) that returns `session.user` as `data`. Shares the session cache entry, so no extra request is made.

## Usage

```tsx
import { useAuth, useUser } from "@better-auth-ui/react"

const { authClient } = useAuth()
const { data: user } = useUser(authClient)
```

Accepts the same arguments as `useSession` — see its [Params](/docs/react/queries/session#params).
