---
title: useCancelInvitation
description: Cancel an organization invitation.
---

Organization mutations require `organizationClient()` on the Better Auth client.

## Usage

```tsx
import { useAuth, useCancelInvitation } from "@better-auth-ui/react"

const { authClient } = useAuth()
const mutation = useCancelInvitation(authClient)

mutation.mutate(/* params */)
```

## Options factory

```tsx
import { cancelInvitationOptions } from "@better-auth-ui/react"
import { useMutation } from "@tanstack/react-query"

const mutation = useMutation(cancelInvitationOptions(authClient))
```
