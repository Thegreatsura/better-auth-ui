## Prerequisites

Before you start, make sure you have the following prerequisites integrated into your project:

- [Better Auth](https://www.better-auth.com/docs/installation)
- [shadcn/ui](https://ui.shadcn.com/docs/installation)
- [Sonner](https://ui.shadcn.com/docs/components/sonner)

## Installation

<Steps>
<Step>
### Install the components

Install the `shadcn/ui` components using shadcn CLI.

```npm
npx shadcn@latest add https://better-auth-ui.com/r/auth.json
```
</Step>
</Steps>
