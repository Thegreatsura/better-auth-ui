title: <OrganizationsSettings />
description: Solid/Zaidan settings panel for organization creation and switching.
---

`OrganizationsSettings` is mounted by the local `organizationPlugin()` in the `/settings/organizations` tab. It lists the current user's organizations, includes the create-organization form used by the example, and renders `UserInvitations` so users can accept or reject invitations sent to their account.

## Usage

```tsx
import { OrganizationsSettings } from "@/components/auth/organization/organizations-settings"

<OrganizationsSettings />
```

## Installation

```npm
npx zaidan add https://better-auth-ui.com/r/solid/organization.json
```

## Composition

`OrganizationsSettings` composes `<Organizations />` and `<UserInvitations />` for the user-level organizations settings tab. Use `<Organization />` for the slug-aware organization management shell with settings and people tabs.
