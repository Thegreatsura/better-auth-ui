## Installation

```npm
npx shadcn@latest add https://better-auth-ui.com/r/solid/sign-in.json
```

After install, `src/components/auth/sign-in.tsx` and the copied sign-in files are app-owned. Register plugins such as Username or Magic Link in `AuthProvider` when you want them to alter the sign-in flow.
