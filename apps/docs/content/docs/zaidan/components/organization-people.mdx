`OrganizationPeople` is the shell for the `/organization/$slug/people` route. It includes a basic members list backed by `useListOrganizationMembers`, permission-gated member role editing and removal for non-current members, an `InviteMemberDialog` for inviting members, and a basic invitations list backed by `useListOrganizationInvitations`.

## Usage

```tsx
import { OrganizationPeople } from "@/components/auth/organization/organization-people"

<OrganizationPeople />
```

## Installation

```npm
npx zaidan add https://better-auth-ui.com/r/solid/organization.json
```

## Members

The members section renders a basic members list with loading and empty states. It also opens `InviteMemberDialog` so admins can invite a member with a selected role, lets permitted users update member roles, and lets permitted users remove non-current members. Full members tables and self-service actions remain deferred: search, filters, sorting, and leave organization are follow-up slices.

## Invitations

The invitations section renders a basic invitations list with loading and empty states. Users with permission can cancel pending invitations. Full invitations tables and actions remain deferred: search, filters, sorting, and advanced status actions are follow-up slices.

## Notes

The People shell exists and is installed with the Organization registry entry. Advanced members and invitations tables and actions remain deferred to follow-up slices.
