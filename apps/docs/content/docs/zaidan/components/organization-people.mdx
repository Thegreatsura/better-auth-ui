`OrganizationPeople` is the shell for the `/organization/$slug/people` route. It now includes a basic members list backed by `useListOrganizationMembers` and keeps invitations ready for a follow-up slice.

## Usage

```tsx
import { OrganizationPeople } from "@/components/auth/organization/organization-people"

<OrganizationPeople />
```

## Installation

```npm
npx zaidan add https://better-auth-ui.com/r/solid/organization.json
```

## Members

The members section renders a basic members list with loading and empty states. Full members tables and actions remain deferred: search, filters, role editing, remove member, and leave organization are follow-up slices.

## Invitations

The invitations section is present as a shell. Invitation tables, invite, cancel, search, filters, and status actions remain deferred.

## Notes

The People shell exists and is installed with the Organization registry entry. Advanced members and invitations tables and actions remain deferred to follow-up slices.
