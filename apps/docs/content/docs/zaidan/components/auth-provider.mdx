## Usage

Install the provider registry payload with `zaidan add https://better-auth-ui.com/r/solid/auth-provider.json`; it copies `src/components/auth/auth-provider.tsx` and related provider support files. After install, these files are app-owned. Then wrap your application shell as shown in the Solid/Zaidan example.

```tsx file=<rootDir>/../../examples/start-solid-zaidan-example/src/components/providers.tsx
```
